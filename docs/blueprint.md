# **App Name**: ResumeKeeper

## Core Features:

- User Authentication: Secure user registration and login to manage personal resume data.
- Profile Section Management: Allow users to input and update 'Basics', 'Experience' (with exactly 3 responsibilities per job), and 'Education' details. Includes validation.
- AI-Powered 'About' Generation: Generate an 80-120 word 'About' paragraph and a 1-2 sentence short bio using a generative AI tool, based on user-provided keywords or job experience.
- Project Portfolio Management: Enable users to add projects, either by URL or file upload, ensuring each project has at least one evidence source. Includes full edit/update flows.
- Publishing and Live Preview: Provide a clean, professional, and accessible layout to preview the complete resume/portfolio and publish it as a public-facing URL.
- Persistent Data Storage: Securely store all user profile and project data to enable persistent resume management across sessions.

## Style Guidelines:

- Primary color: A confident and professional medium-dark blue, Hex: #1E52CC. This hue (HSL: 220, 80%, 40%) conveys trust and professionalism while being sufficiently dark to stand out on a light background.
- Background color: A very light, desaturated cool grey, Hex: #ECEFF2. This background (HSL: 220, 15%, 95%) ensures optimal readability and provides a clean canvas.
- Accent color: A vibrant yet clean turquoise, Hex: #32BFDB. This color (HSL: 190, 70%, 55%) offers an energetic contrast to highlight interactive elements and calls to action.
- All text will use 'Inter', a grotesque-style sans-serif font. Its modern, objective, and neutral design ensures high legibility and a professional appearance for both headlines and body text.
- Use a set of simple, outline-style icons that are consistent in weight and form. Prioritize clarity and intuitive recognition to support a clean user interface.
- Implement a clean, spacious, and grid-based layout, utilizing ample whitespace to enhance readability and visual hierarchy, aligning with modern and accessible UI principles. Ensure responsiveness across devices.
- Incorporate subtle and functional animations for UI feedback, such as smooth transitions between sections and mild hover effects, to improve user experience without being distracting.
